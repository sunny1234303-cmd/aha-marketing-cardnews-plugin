# Card News Creator — by Aha Marketing

Aha Marketing에서 생성한 카드뉴스 카피를 Figma 캔버스에 **디자인된 슬라이드**로 자동 삽입하고
**PNG 파일**로 바로 내보낼 수 있는 플러그인입니다.

> 기존 [Markdown to Figma 플러그인](../15-figma-md-plugin/)과 별개로 운영됩니다.
> 기존 플러그인: 문서형 텍스트 삽입 / 이 플러그인: 디자인된 카드뉴스 생성

## 설치 (최초 1회)

1. **Figma 데스크탑 앱** 실행 (웹 버전 불가)
2. 상단 메뉴 → `Plugins` → `Development` → `Import plugin from manifest...`
3. 이 폴더의 `manifest.json` 선택

## 사용 방법

1. `Plugins` → `Development` → **Card News Creator by Aha Marketing** 실행
2. **슬라이드 추가** 후 각 슬라이드 편집:
   - **제목**: 큰 볼드체로 표시 (필수)
   - **본문**: 부제목/설명 (선택)
   - **태그/뱃지**: Enter로 추가 (서비스 기획, GTM 등)
   - **배경**: 색상 선택 또는 이미지 업로드
   - **이미지 배경 시**: 오버레이 어둡기 조절
   - **텍스트 색상 / 뱃지 강조색** 커스텀
3. **설정 탭**에서 폰트·프레임 크기·글자 크기 조정
4. **Figma에 삽입** → 캔버스에 슬라이드 프레임 생성
5. **PNG 내보내기** → 각 슬라이드를 PNG 파일로 다운로드 (2x 해상도)

## 지원 기능

| 기능 | 설명 |
|------|------|
| 슬라이드 구성 | 제목 / 본문 / 태그 뱃지 / 배경 |
| 배경 | 단색 색상 or 이미지 업로드 + 오버레이 |
| 프레임 크기 | SNS 1080×1080 / 스토리 1080×1920 / 자유 설정 |
| 폰트 | Figma에 설치된 모든 폰트 (Pretendard, Noto Sans KR 등) |
| 슬라이드 조작 | 순서 변경 ↑↓ / 복제 ⧉ / 삭제 ✕ |
| 슬라이드 번호 | 다중 슬라이드 시 우상단 자동 표시 |
| PNG 내보내기 | 2x 고해상도, 삽입 직후 바로 다운로드 가능 |
